Execute PHP inside posts, pages and text widgets.

== Description ==

Execute PHP inside posts, pages and text widgets. Use [exec][/exec] to execute PHP.

Inspired by [PHP Exec](http://priyadi.net/archives/2005/03/02/wordpress-php-exec-plugin "") by Priyadi Iman Nurcahyo.

**NOTE:**
Always use HTML view in the editor as Visual escapes text and destroys the PHP.

== Installation ==

1. Upload `/wp-exec-php/` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
